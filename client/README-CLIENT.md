# Client

## Install:
### Make sure to use the same .env values as in the server!
`Rename .env.local.template to .env.local and fill in the values.`

`Change also Port in `[vite.config.ts](vite.config.ts)

`npm install`

## Start Client:
`npm run client`

http://localhost:4200/

## Start StoryBook:
`npm run storybook`

http://localhost:6006/
